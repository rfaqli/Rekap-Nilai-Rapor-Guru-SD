# Katrol Nilai

![status](https://img.shields.io/badge/status-MVP-007acc?style=flat-square) ![built with](https://img.shields.io/badge/built_with-React_|_Node.js_|_PostgreSQL-e34f26?style=flat-square) ![target](https://img.shields.io/badge/target-Teachers-ffc107?style=flat-square) ![focus](https://img.shields.io/badge/focus-Productivity-4caf50?style=flat-square)

Alat Bantu Rekap & Kalkulasi Nilai Rapor Otomatis yang dirancang untuk memudahkan para guru.

## Project Purpose

Proyek ini bertujuan untuk menyediakan pengalaman yang efisien dan menghemat waktu bagi para guru melalui kalkulasi nilai otomatis, penyimpanan data yang terstruktur, dan rekapitulasi nilai siswa dengan mudah.

The website is designed to be:
- Easy to use
- Visually clean and professional
- Secure with cloud database integration to save projects safely

## Target Users

- Teachers and educators
- School administrators
- Anyone who needs to process student grades quickly

## Main Features

- Secure user authentication (Login & Register)
- User-specific Dashboard for managing projects
- Admin Dashboard to view registered users
- Automatic grade calculations and processing
- Persistent data storage

## Technologies Used

- React (TypeScript) – Frontend structure and UI
- Tailwind CSS – Professional and clean styling
- Express.js – Backend server logic
- PostgreSQL & Drizzle ORM – Relational database management
- JWT & bcrypt – Secure authentication and authorization

## Project Status

- Version: MVP (Minimum Viable Product)
- Development: Ongoing
- Database: PostgreSQL

## Future Improvement

- Add export to Excel and PDF formats
- Improve UI responsiveness on mobile devices
- Add more advanced grading formulas and parameters
- Data visualization for student performance
